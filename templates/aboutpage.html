About This Project

Project Title: Voice-Controlled IDE for People with Disabilities

Team Members: Divya Dharshini V, Harine Selvaraj, Alwin Dharsan

Mentor: Dr. M. Vimaladevi, Associate Professor

The Problem

  • Lack of accessible coding environments for people with motor impairments.
  • Existing tools are complex, expensive, or not beginner-friendly.

The Solution

  • Voice to Code using natural language
  • In-browser IDE with NLP & accessibility tools
  • Font resizing, dark mode, voice-only operation
